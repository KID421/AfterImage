# AfterImage
Unity上で3Dモデルの残像処理を行うサンプルプロジェクト

* Windows10
* Unity2019.4.11f1 / URPプロジェクト
#### 主な追加パッケージ
* Cinemachine(2.3.4)
* UniRx(7.1.0)/UniTask(2.0.37)
